# Amazon E-commerce EDA

Датасет с продажами индийского Amazon за март-июнь 2022, около 120к заказов. 
Хотел разобраться что происходит с бизнесом и найти проблемные места.

Из интересного: выручка падает на протяжении всего периода, при этом 
merchant-продавцы отменяются в 2 раза чаще чем Amazon (13% vs 6%), 
и почти все отмены происходят до отправки - то есть проблема не в логистике, 
а раньше. Ещё Set и Kurta дают 75% выручки, остальные категории почти не влияют.

Дашборд в Tableau — выручка по дням, категории, отмены по методу доставки, топ штатов.

Данные: [Kaggle](https://www.kaggle.com/datasets/thedevastator/unlock-profits-with-e-commerce-sales-data)



# Facebook Bidding A/B Test

Facebook тестировал два типа биддинга — maximum и average. 
Данные за 40 дней, нужно было понять какой работает лучше.

Сначала проверил корректность дизайна: AA тест прошёл нормально, 
данные распределены нормально по Шапиро-Уилку

По результатам: average bidding даёт на 32% больше выручки и на 18% больше показов, 
при этом кликов меньше но это не страшно — конверсия та же. 
По покупкам разница есть (+5.7%) но статистически не значима — для вывода нужно минимум 373 наблюдения вместо 40. 
Применил поправку Бонферрони на множественное тестирование — выводы не изменились.

Данные: [Kaggle](https://www.kaggle.com/datasets/furth3r/facebook-ab-test-of-bidding-feature)
